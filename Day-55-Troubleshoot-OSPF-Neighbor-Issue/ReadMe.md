# Day 55 - Troubleshoot OSPF Neighbor Issue

## Objective

Identify and resolve OSPF neighbor failures.

## Root Causes

- Area Mismatch
- Timer Mismatch
- Passive Interface
- Wrong Network Statement
- Interface Down

## Commands

show ip ospf neighbor
show ip ospf interface
show ip route ospf

## Result

Successfully restored OSPF adjacency and route exchange.
