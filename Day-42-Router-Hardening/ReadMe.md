# Day 42 - Router Hardening

Secure a Cisco router using enterprise security best practices.

## Security Features

- Password Encryption
- SSH Only Access
- Login Banner
- Login Blocking
- Secure Console
- Secure VTY
- Disabled Unused Interfaces

show login
show ip interface brief

Successfully hardened router against common management and access threats.
